## Stack utilizada

**Front-end:** React, Bootstrap, CSS

**Back-end:** Node, Express e Mysql


## Deploy

Para reproduzir o projeto, abra o terminal e insira:


```bash
  cd front 
```
```bash
  npm run dev
```

Para rodar o backend, insira os comandos:

```bash
  cd backend
```
```bash
  npm start
```
